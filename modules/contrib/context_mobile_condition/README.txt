INTRODUCTION
--------

This is a Context Mobile Condition module which integrates Context and PHP Mobile Detect library Mobile_Detect.

Context Mobile Condition module is used for detecting devices such as mobile, tablet and computer as per the condition. It uses the user-agent string combined with specific HTTP headers to detect the mobile device type
environment. This module can detect multiple device at a time by using checkbox condition.

* For a full description of this module, visit the project page:
   https://www.drupal.org/project/context_mobile_condition

* To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/context_mobile_condition

REQUIREMENTS
------------

This module requires the following module and library:

* Context (https://www.drupal.org/project/context)
* Mobile_Detect (https://github.com/serbanghita/Mobile-Detect)

INSTALLATION
------------

To install Context Mobile Condition module, it requires Context module and Mobile_Detect library. Then drop it into your modules folder and turn it on and enable the required module.

For Implementing this module:
Search Context under the Structure menu and click on it, you will find a form where you want to enter Label, Group and Description.

Enter the "Conditons" for which device you want to detect and add "Reaction" such as block.

USAGE
-----

Once module is installed you are getting new Context Conditions, which are currently:

- Mobile Device (mobile, tablet, desktop).

Those are conditions you might used to detect the mobile device.
