INTRODUCTION
------------

Sajari provides powerful, custom search solutions for websites.

This module will provide easy to configure blocks and templates
to help integrate Sajari into Drupal.

* For a full description of this module, visit the project page:
   https://www.drupal.org/project/sajari

* To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/sajari

REQUIREMENTS
------------

This module requires the following module:

* JSON Template (https://www.drupal.org/project/json_template)

INSTALLATION
------------

Install/Enable the sajari module as you would normally
install a contributed Drupal module.

* Visit (https://www.drupal.org/node/1897420) for further information.

* Optionally you can also install using composer:
   composer require 'drupal/sajari:1.0.x-dev'

CONFIGURATION
-------------

Once module is installed then:

* Go to 'Configuration -> Search and metadata -> Sajari' and have found
two options that are 'Project' and 'Collection'.
